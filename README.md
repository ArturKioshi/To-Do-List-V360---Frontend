# V360 To Do List - Backend

Este é o frontend da aplicação de gerenciamento de tarefas (To-Do List) desenvolvida para o desafio técnico da V360. A Interface foi construída utilizando React com Typescript.

## Backend

Para esse projeto, é consumida a API disponível no repositório: [To-Do-List-V360 - Backend](https://github.com/ArturKioshi/To-Do-List-V360---Frontend)

## Tecnologias
- React 19.2.0 com TypeScript
- Vite
- Tailwind CSS
- React Query (TanStack Query) e Axios
- Phosphor React

## Instalação e Uso

- Clone o repositório
```
git clone url_do_repositorio
cd frontend
```

- Instale as dependências

```
npm install 
```

Crie um arquivo .env na raiz do projeto e aponte para a URL do backend

```
VITE_API_URL=http://localhost:3000/api/v1
```

- Rodar o servidor

```
npm run dev
```
